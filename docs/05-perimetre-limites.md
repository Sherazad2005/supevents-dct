# Périmètre et limites

## Périmètre fonctionnel

| Dans le périmètre | Hors périmètre |
|---|---|
| Création d’événements étudiants | Application mobile native |
| Gestion des réservations | Support hors ligne |
| Génération de billets numériques | Migration de données depuis un ancien système |
| Authentification des utilisateurs | Support d’Internet Explorer 11 |
| Gestion des rôles utilisateurs | Intégration avec des réseaux sociaux |
| Paiement en ligne | Gestion d’événements externes à l’établissement |
| Consultation des événements | Système de visioconférence intégré |

## Limites techniques définies par l’équipe

| Limite technique | Justification |
|---|---|
| Migration de données existantes hors périmètre | Le projet est conçu comme une nouvelle plateforme sans base de données existante. |
| Support d’Internet Explorer 11 hors périmètre | Navigateur obsolète non adapté aux technologies web modernes utilisées par SupEvents. |
| Application mobile native non prévue | Priorité donnée à une application web responsive accessible sur tous les navigateurs récents. |
| Hébergement local non supporté | Le projet vise un déploiement cloud simplifiant la maintenance et la scalabilité. |
